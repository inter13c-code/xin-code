# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT open a public issue**
2. Email: [Create a private security advisory](https://github.com/xincode-ai/xin-code/security/advisories/new)
3. Include: description, reproduction steps, and potential impact

We will respond within 48 hours and work with you to resolve the issue before public disclosure.

## Scope

- API key / credential handling
- Command injection via tools (Bash, Edit, Write)
- Permission system bypass
- MCP server security
- Session data exposure

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | Yes       |
| < latest | No      |
